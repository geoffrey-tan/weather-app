## :ledger: Version Control

#### Feature

```shell
git checkout -b feature/<name>
```

_OR_

```shell
brew install git-flow-avh
```

```shell
git flow feature start <name>
```

```shell
git flow feature finish
```

#### Commit

```shell
npm run cz
```

_OR_

```shell
npm i -g commitizen
```

```shell
cz
```

_OR_

```shell
code --install-extension KnisterPeter.vscode-commitizen
```
